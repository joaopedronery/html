Principais formatações

Negrito / Destaque

Para escrever palavras em negrito basta utilizar a tag <b>(não semântica)

Nessa frase temos um termo em destaque, utlizando a tag <strong>(semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico, utilizando a tag <i>(não semântica)

Nessa frase temos um termo com ênfase, utilizando a tag <em>(semântica)

Texto marcado

Nessa frase temos um termo marcado usando a tag <mark>

Nessa outra frase também temos um termo marcado

Texto deletado

Podemos marcar um termo como excluído para indicar que ele deve ser lido, mas não considerado. Utilizamos a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizamos a tag <ins>

Texto sobrescrito

Para inserir coisas do tipo: x20 + 3. Utilizamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo h2o. Utilizamos a tag <sub>
